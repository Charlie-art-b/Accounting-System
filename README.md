📊 Accounting System

A robust accounting system built with Laravel (PHP) using the MVC architecture and Filament for the admin panel. This application provides essential financial management tools for business operations.

🚀 Features

General Ledger (Journal Entries)

Accounts Receivable

Accounts Payable

Inventory Management

Financial Reporting

User Management with Roles & Permissions (RBAC)

🛠️ Tech Stack

Backend: Laravel (PHP)

Admin Panel: Filament

Database: MySQL

ORM: Eloquent

🔐 Security

Role-Based Access Control (RBAC)

Authentication and authorization handled within Laravel ecosystem
